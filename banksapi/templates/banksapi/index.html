Indian Banks REST API

  • List of banks : http://indianbanksrestepc.herokuapp.com/banks
  • List of bank branches : http://indianbanksrestepc.herokuapp.com/banks/branches
  • To search for a bank with ‘id’(not official-provided by the database) : http://indianbanksrestepc.herokuapp.com/banks/‘id’
  • To search for a bank branch with with IFSC code : http://indianbanksrestepc.herokuapp.com/banks/branches/ifsc=‘ifsc_code’
  • To search for all the branches of a particular bank in a particular city : http://indianbanksrestepc.herokuapp.com/banks/branches/name=‘bank_name’&city=‘city_name’
